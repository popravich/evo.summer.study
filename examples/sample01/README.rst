Пример #1. Структура проекта
============================


.. code-block:: bash

   /sample01
      /templates/
         /base.html
         /...
         /...
      /__init__.py
      /__main__.py
      /config.py
      /main.py
      /views.py


``sample01`` -- Имя проекта, python package.

``config.py`` -- Содержит функции для конфигурации Flask-приложения и проекта в целом.

``views.py`` -- Содержит view-функции.
